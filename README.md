1. 일주일 5문제 중 3문제 이상 풀기 [월-일] <br/><br/>
2. 언어
   - 김승겸 : Python
   - 선재우 : C++, Python
   - 최성우 : Python
<br/><br/>
   
3. 벌금

   - 1회당 1000원
<br/><br/>

4. 정기 모임

   - 매주 화요일 21시 **3문제씩 정하기** 
   - 코드리뷰는 차후 결정
<br/><br/>   
   
5. 문제를 풀면 백준 번호와 Readme 작성
